# AI-Enabled IoT Plant Care System 🌱

An intelligent IoT-based plant monitoring system built using ESP8266 that continuously tracks soil moisture, temperature, humidity in real time. Sensor data is transmitted to Firebase and visualized through a web dashboard for remote monitoring.

The system incorporates a machine learning model (Random Forest) to predict plant health status and an AI chatbot that provides personalized plant care recommendations based on live sensor data. This project demonstrates sensor interfacing, embedded firmware development, cloud integration, and AI-assisted decision making for smart agriculture applications.

## Features

* Real-time monitoring of soil moisture, temperature, humidity.
* ESP8266-based firmware for sensor data acquisition
* Firebase cloud database integration
* Web dashboard for remote visualization
* Random Forest-based plant health prediction
* AI chatbot for personalized care recommendations
* Threshold-based alerts and recommendations

## Technologies Used

* ESP8266
* Arduino IDE (Embedded C/C++)
* Firebase
* Python
* Random Forest Machine Learning
* HTML, CSS, JavaScript
* IoT and Cloud Integration
